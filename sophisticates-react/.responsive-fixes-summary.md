## Key Techniques Used

### 1. CSS clamp() Function
Used throughout for fluid typography and spacing:
```css
font-size: clamp(min, preferred, max)
padding: clamp(min, preferred, max)
```

### 2. CSS Custom Properties
Dynamic values that change per breakpoint:
```css
--section-spacing: clamp(100px, 12vw, 160px)
--container-padding: clamp(24px, 5vw, 64px)
```

### 3. Utility Classes
- `.grid-stack-mobile` - Stacks grids on mobile
- `.grid-stack-tablet` - Stacks grids on tablets
- `.mobile-no-border` - Removes borders on mobile
- `.flex-col-mobile` - Changes flex direction on mobile
- `.desktop-only` - Hides on mobile
- `.mobile-only` - Shows only on mobile

### 4. Responsive Grid
```css
grid-template-columns: repeat(auto-fit, minmax(min(100%, 300px), 1fr))
```

### 5. Viewport Units
Used strategically for spacing:
```css
padding: clamp(40px, 8vw, 100px)
gap: clamp(40px, 10vw, 100px)
```
